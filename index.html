About This Hub
A living library of Snowflake value slides

The Snowflake Value Realization Hub is a searchable catalog of 46 individual value realization slides — one per GitHub repo — each showing how a specific Snowflake solution addresses a real business challenge. Every slide follows the Snowflake 2026 brand template and is ready to drop into a customer conversation, QBR, or executive presentation.

The hub is maintained by the RevOps and SE team and is designed to grow over time. Each slide maps business challenges to Snowflake capabilities, quantifies outcomes, and recommends concrete next steps.

  • Snowflake-branded slides ready for customer-facing use
  • Each slide maps a specific challenge to Snowflake capabilities and outcomes
  • Filter by persona: Sales Engineering, Revenue Operations, or Analytics
  • Click any slide to open the full rendered version
  • Every slide links back to its GitHub source repo
  • Feedback form to request new slides or improvements

How It Works
Three steps from catalog to conversation
1

Browse & Filter

Search 46 solution slides by persona, capability (Cortex AI, Streamlit, dbt, etc.), or keyword. Persona filter buttons narrow to your audience instantly.

2

Open the Slide

Click "View Slide" to open the full Snowflake-branded HTML slide — challenge, capabilities, outcomes, and next steps in one screen. No login required.

3

Share or Extend

Share the slide link directly, or navigate to the GitHub repo to see the full README, documentation, and source code behind the solution.

Contributing
Add your solution to the Hub

The catalog grows through team contributions. If you have a Snowflake solution, tool, or workflow documented in a GitHub repo, the value-realization-slide CoCo skill generates a branded slide for it in seconds.

value-realization-slide
CoCo Skill

A Cortex Code skill that reads any GitHub repo's documentation, analyzes the challenge/capabilities/outcomes, and produces a pixel-perfect Snowflake 2026-branded HTML slide — ready to commit and share. No template editing required.

Run against your repo

View Skill on GitHub → Get CoCo
1

Install the Skill

In CoCo, paste the install command above. The skill is ready in seconds — no configuration needed.

2

Run Against Your Repo

Type /value-realization-slide and your GitHub repo URL. The skill fetches your README and generates the branded slide.

3

Commit the Slide

The slide is saved to ~/Downloads. Commit it to your repo root as value-realization.html.

4

Submit to the Hub
